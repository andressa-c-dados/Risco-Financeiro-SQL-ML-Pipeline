# Risco Financeiro.
Risk Intelligence — Pipeline de Análise de Risco Financeiro
Este projeto apresenta uma solução completa para análise de risco financeiro, desenvolvida originalmente durante o Hackathon Elas + Tech. O pipeline integra desde a qualidade e tratamento de dados até a validação cruzada de modelos preditivos.

🛠️ Tecnologias e Metodologias
SQL & Python: Utilizados para manipulação robusta de dados e modelagem.

Machine Learning: Comparação de múltiplos modelos preditivos para Score de Crédito.

Validação Robusta: Implementação de validação cruzada para garantir a confiabilidade dos resultados.

Governança: Foco em qualidade de dados e mitigação de viés.

📂 Organização do Repositório
/SQL: Contém o notebook principal (Pipeline_unificada_Risco_Financeiro.ipynb) com o fluxo consolidado de dados e modelos.

README.md: Documentação técnica e visão executiva do projeto.
